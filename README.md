## Project info
Simple shopping cart created with React.

## Technologies
Project is created with:
* Vite: 3.2.3
* React: 18.2.0
* React-Redux: 8.0.5
* Redux Toolkit: 1.9.1
* Tailwind CSS: 3.2.4
* Typescript: 4.6.4
	
## Setup
To run this project, install it locally using npm:

```
cd ../table-exercise
npm install
npm run dev
```
